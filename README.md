# network-traffic-checksum-analysis
This repository contains network traffic analysis programs
using Python Scapy and Wireshark checksum validation.

Protocols Covered:
- IP
- TCP
- UDP
- ICMP
- Ethernet
- TLS

How to Run

1. Install dependencies:
   pip install scapy
  

3. Run any protocol script:
   python icmp.py
   python tcp_cs.py
   python tcp_cs.py
   python udp_cs.py
   python eth.py
   python tls.py
